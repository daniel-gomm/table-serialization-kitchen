tableserializer.integrations module
====================================

Submodules
----------

To use the integrations, the package has to be installed with extras.

tableserializer.integrations.target module
------------------------------------------

To use the `TARGET <https://target-benchmark.github.io/>`_ integration, install table serialization kitchen with the
TARGET integration:

.. code-block::

   pip install tableserializer[target]
