class tableserializer.serializer.serializer.Serializer(recipe: SerializationRecipe, metadata_serializer: MetadataSerializer | None = None, schema_serializer: SchemaSerializer | None = None, table_serializer: RawTableSerializer | None = None, row_sampler: RowSampler | None = None, table_preprocessors: List[TablePreprocessor] | None = None)[source]

Bases: object

Serializer that serializes a given table according to a user-specified format.

Parameters:

  • recipe (SerializationRecipe) – The recipe detailing the serialization.

  • metadata_serializer (MetadataSerializer) – Serializer for the table metadata. Only needed if there is metadata placeholder in the recipe.

  • schema_serializer (SchemaSerializer) – Serializer for the table schema. Only needed if there is schema placeholder in the recipe.

  • table_serializer (RawTableSerializer) – Serializer for the raw table. Only needed if there is table placeholder in the recipe.

  • row_sampler (RowSampler) – Optional module that samples a number of rows from the raw table before serializing.

  • table_preprocessors (List[TablePreprocessor]) – Optional list of table preprocessors that transform the table before serialization.
